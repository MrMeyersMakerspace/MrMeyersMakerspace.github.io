Object Creation

Objects are created using very similar syntax as other programming languages.


An object for a person was created using the following code:

var person = {
    firstname  : "John",
    lastname   : "Henry",
    age        : 33,
    occupation : "Steel Driving Man"
};


Accessing Object Properties

Object properties are accessed in one of two ways, either using "dot notation" or with brackets. For example we can access the age of a person in a person object using either person.age or person["age"].

Object Methods

Methods can be created for objects just like properties can be created. They would need to be defined at time of creation. The method is called with the name of the method and (). Member properties can be accessed using the this keyword like in many programming languages. Below you will find an example of the previous object being created but now with a method that will display the full name of the person:


var person = {
    firstname  : "John",
    lastname   : "Henry",
    age        : 33,
    occupation : "Steel Driving Man",
    fullName   : function() {
      return this.firstname + " " + this.lastname;
    }
};

We can also add a method to an existing object just like we did a property using the following code:

person.fullName = function() {
    return this.firstname + " " + this.lastname;
}

Object Accessors (Setters and Getters)

We can set and access data using setters and getters, just like in other programming languages. These have the advantage of being more secure and have simplier syntax that looks like properties instead of functions. To change our fullName function to a getter the code would look like the following:

get fullName() {
    return this.firstname + " " + this.lastname;
}

To access this getter we now use person.fullName instead of the function person.fullName(). A setter has very similar syntax. To create a setter to change an objects property it would like this:

set firstname(fname) {
    this.firstname = fname;
}

To set a property you would then use a command like person.firstname = "George".


Object Constructors

We can create a constructor for our objects so that you can easily create new objects and define their properties. This time we will create a new car object with constructors as well as the setters and getters:

function car(make,model,color) {
    this.make = make;
    this.model = model;
    this.color = color;
};
var geo = new car("Geo", "Prism", "Black");
