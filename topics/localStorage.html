Local Storage

There are 5 main methods for localStorage that come in handy. Those are:

  1. setItem()
  2. getItem()
  3. removeItem()
  4. clear()
  5. key()

setItem()

The method setItem() is used to store values in a localStorage object. Below you will enter a value that will be stored in local storage. It takes two parameters, a key and a string value.

Enter a value and key below and click "Store Value" to store the pair in localStorage.




Code: window.localStorage.setItem(key, value);



getItem()

The method getItem() allow you to access the localStorage object and retrieve data. You use the specific key for the item you want and it returns it as a string. It takes one parameter, the key.

Enter the key and click on the button below to display the stored value from local storage


Value:


Code: window.localStorage.getItem(key);



removeItem()

The method removeItem() is used for removing a value from the localStorage object. If this is called and there isn't anything stored with that key then it does nothing. It takes one parameter, the key.

Enter a key below to remove that item from the localStorage object.



Code: window.localStorage.removeItem(key);



clear()

The method clear() is used to remove all data from localStorage. By running this method it clears out all data. It takes no parameters.

Press the button below to remove all data from localStorage.


Code: window.localStorage.clear();



key()

The method key() is used to find out the key for a specific string in the localStorage object. It takes one parameter, an index.

Code: window.localStorage.key(index);
