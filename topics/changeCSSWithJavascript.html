Manipulating CSS Class Properties Using JavaScript

JavaScript can be used to change the CSS properties of various elements. The syntax to make it happen is extremely simple. All you have to do is call the item using DOM and then add style.propertyName to the end and then set it equal to the desired value. One key difference is the property name will be written in camelCase instead of snake_case.
